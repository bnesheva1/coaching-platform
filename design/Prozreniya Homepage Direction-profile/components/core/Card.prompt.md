Card — content container for value props, practitioner summaries, and homepage sections. `tone="inverse"` flips to the dark surface for contrast blocks inside a light page (or vice versa).

```jsx
<Card eyebrow="За търсещите" title="Намерете яснота" description="Проверени практици, реални отзиви." footer={<Button variant="secondary" size="sm">Разгледайте</Button>} />
```
