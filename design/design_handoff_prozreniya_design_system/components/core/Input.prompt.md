Input — text entry. Set `search` for the hero/marketplace search bar (rounded pill, embedded gold go-button); omit for standard form fields.

```jsx
<Input search placeholder="Търсете по име, направление или въпрос…" helperText="Търсете по име, по направление или по вашия въпрос" />
<Input placeholder="Имейл" />
```

Focus state: gold border + soft focus ring (`--focus-ring`), never a browser default outline.
